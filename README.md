
ganjiuwanle.com — 简易中文承诺站（本地演示）

说明
- 这是一个最小可运行的示例，将英文站风格改成中文并提供本地表单展示（不含支付或后端持久化）。
- 将项目放到支持 PHP 的环境运行即可。

本地运行
1. 在项目根目录运行：
```bash
php -S localhost:8000
```
2. 打开浏览器访问 `http://localhost:8000`

注意与扩展建议
- 当前实现仅在页面上显示提交结果，并不会保存到数据库。如需保存请接入 SQLite / MySQL 或 REST API。
- 若要接入支付（例如 Stripe、支付宝），请在表单提交后启动支付流程，并在支付成功后记录承诺为有效。
- 若你想把站点上线到 `ganjiuwanle.com`：
	- 注册域名 `ganjiuwanle.com`（如果尚未购买）并将 DNS 指向你的主机。
	- 在云主机或 VPS 部署 PHP/NGINX 或使用静态托管 + 后端 API。
